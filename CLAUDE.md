# Daniel Puig Gerarde Consulting Website

Daniel's personal website pivoted from an AI SRE/Agent focus to a premium **Professional Consulting in Software Architecture, The Cloud, and DevOps** with a focus on foundational platforms such as **Kubernetes and its entire ecosystem**.

## Tech Stack
- **Framework**: Next.js 14 (App Router, MDX)
- **Styling**: Tailwind CSS 4.x
- **Key Libraries**: Headless UI, @next/mdx, fast-glob

## Development Commands
- Run development server: `npm run dev`
- Build production version: `npm run build`
- Run linting: `npm run lint`

## Design & Architecture Philosophy
- Curated color palettes with a high-converting, professional, and sophisticated look.
- Core Pillars: Resilient Architecture, Scale-ready Cloud Platform engineering, Kubernetes zero-trust environments, GitOps automation.
- Focus on performance, clean semantic HTML, and fully responsive layouts.
